# WatchTogether - Local Setup

## Requirements
- Node.js 18+
- MongoDB (local ya MongoDB Atlas)

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on: http://localhost:5000

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on: http://localhost:5173

---

## MongoDB

Local MongoDB install hai toh seedha chalega.
Atlas use karna hai toh backend/.env me MONGODB_URI update karo.
